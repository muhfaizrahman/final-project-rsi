# Lunaris
Lunaris adalah aplikasi web untuk pencari kerja (pelamar) dan perusahaan. Fitur utama: manajemen lowongan, manajemen profil pengguna/perusahaan, membaca artikel, memberi komentar pada artikel, bookmark (menyimpan) lowongan.  
Projek ini merupakan pemenuhan tugas akhir mata kuliah Rekayasa Sistem Informasi kelas E
