# Lunaris
Lunaris adalah aplikasi web untuk pencari kerja (pelamar) dan perusahaan. Fitur utama: manajemen lowongan, profil pengguna/perusahaan, chat, event, artikel, bookmark, dan aplikasi pekerjaan.  
Projek ini merupakan pemenuhan tugas akhir mata kuliah Rekayasa Sistem Informasi kelas E
